1. Explore Apollo - 09/01/15 to 12/11/15

Project Summary

o This is a senior design project which aims to create a system providing access to digitized audio files from NASA mission 11. The entire application is split into 4 modules: front-end web page, API server, audio control server and database.
o Our team is comprised of 4 UTD undergrads including me. We were expected to set up the basic development environment and tackle some critical use cases, as the next team will carry on with what we came up with, since this is a 2-semester project.
o The entire system is open source: https://github.com/UTD-CRSS

What I did

o Building the API server using NodeJS, Express framework and PostGreSQL.
o Discussing with the front-end to define critical use cases.
o Designing and exposing API endpoints to the front-end.
o Writing sql scripts to query the database.
o Giving input in designing the database schema and prototyping front-end UIs.

What I learnt

o Basic knowledge of designing RESTful API server.
o Using nodeJS and ExpressJS.
o Revisiting sql scripts.
o Revisiting relational database concepts, basic operations and database schema.
o Helping with prototyping front-end UIs.

Technologies and Softwares used

NodeJS, ExpressJS, PostGreSQL and Git

2. XO_316 - 09/01/15 to 12/11/15

Project Summary

o This is a final 2D game project in my Intro to Video Game Programming Class, using Unity Engine.
o Gameplay is based off Bomberman 1993, where the player controls a character capable of placing an auto-detonate bomb to destroy monsters and objects. The player wins when he/she defeats the finall boss.
o For art design, I used PopTag! as my reference.

What I did

o Sketching and designing several art components including: the character, blocking objects, background, and detonation sprites
o Gameplay implementation including critical mechanics: bomb explosion, character movement, monster attacks and movements.
o Animating moveable objects: character and the monsters

What I learnt

o Unity API and Physics2D
o Using Unity API to animate objects.
o Learning C#
o Basic concepts and expectation in art design

Technologies and Softwares used

Unity3D, Git, Photoshop and Inkscape
